# Clima App

Aplicação web de consulta de clima desenvolvida com HTML, CSS e JavaScript, consumindo a API da Open-Meteo para exibir dados em tempo real.

## Funcionalidades

- Buscar cidade pelo nome
- Exibir temperatura atual
- Exibir velocidade do vento
- Exibir previsão dos próximos dias
- Interface dinâmica com atualização em tempo real

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- API Open-Meteo

## Como executar

1. Clone o repositório:

git clone git@github.com:itsknlll/clima-app.git

2. Abra o arquivo index.html no navegador

## Demo

Acesse a aplicação em funcionamento:

## Objetivo

Este projeto foi criado para praticar consumo de API, manipulação do DOM, uso de async/await e tratamento de requisições assíncronas.

## Créditos

Dados meteorológicos fornecidos pela Open-Meteo API:
https://open-meteo.com/
